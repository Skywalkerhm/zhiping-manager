# 智评管家 - 项目总结

## 📊 项目概览

**智评管家** 是一个帮助大众点评/美团商家高效管理真实用户评论的 SaaS 系统，提供 AI 智能回复、数据分析、多店管理等功能。

- **开发时间**: 2026 年 3 月 17-18 日
- **开发周期**: 2 天
- **代码规模**: 77 个文件，11,300+ 行代码
- **API 接口**: 35 个
- **前端页面**: 5 个
- **数据库表**: 12 张

---

## ✨ 核心功能

### 1. 多平台评论聚合
- 支持大众点评、美团、饿了么
- 统一管理和查看
- 自动同步更新

### 2. AI 智能回复
- 集成 6+ 个 LLM（通义千问、文心一言、腾讯混元等）
- 故障自动切换
- 情感分析回复
- 成本统计

### 3. 深度数据分析
- 评分趋势分析
- 关键词提取（jieba 分词）
- 竞品分析
- 每日报告生成
- ECharts 数据可视化

### 4. 多店管理
- 支持连锁店
- 一键切换店铺
- 统一统计

### 5. 付费订阅
- 4 档订阅计划（Free/Pro/Premium/Enterprise）
- 完整的支付系统
- 订单管理

### 6. 实时通知
- 微信消息推送
- 新评论实时提醒
- 每日报告推送

### 7. 定时任务
- 自动检查新评论（每 10 分钟）
- 自动同步评论（每小时）
- 自动发送日报（每天 9 点）

## 🚀 快速开始

### 后端启动

```bash
# 克隆项目
git clone https://github.com/yourusername/zhiping-manager.git
cd zhiping-manager

# 创建虚拟环境
python3 -m venv venv
source venv/bin/activate  # Windows: venv\Scripts\activate

# 安装依赖
pip install -r requirements.txt

# 配置环境变量
cp config/.env.example .env
# 编辑 .env 文件

# 启动 Docker（MySQL + Redis）
docker-compose up -d

# 初始化数据库
mysql -h 127.0.0.1 -u root -p < scripts/init_db.sql

# 启动应用
uvicorn app.main:app --reload

# 访问 API 文档
open http://localhost:8000/docs
```

### 前端启动

```bash
cd frontend

# 安装依赖
npm install

# 启动开发服务器
npm run dev

# 访问前端界面
open http://localhost:3000
```
